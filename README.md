## 📌 Project Overview

This project is an interactive **Power BI Car Sales Analytics Dashboard** developed during my internship at **Codec Technologies**.

The goal of this project is to transform raw car sales data into meaningful business insights using data analysis and visualization techniques. The dashboard helps stakeholders understand sales performance, customer behavior, and regional trends to support data-driven decision-making.

The dataset was first cleaned and pre-processed using Python, then imported into Power BI for visualization and interactive dashboard development.

## 🎯 Problem Statement

Organizations often struggle to understand:
- Which car brands generate the highest revenue?
- Which regions perform best in sales?
- How customer demographics affect purchasing behavior?
- What are the monthly sales trends?

This project solves these challenges by creating an interactive dashboard for clear and actionable insights.
